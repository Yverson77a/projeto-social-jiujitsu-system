# Sistema de Gestão para Projeto Social de Jiu-Jitsu

Sistema web desenvolvido para auxiliar na administração de um projeto social de jiu-jitsu, permitindo o controle de alunos, avisos, treinos, campeonatos, links úteis e frequência.

## Sobre o projeto

Este sistema foi criado com o objetivo de substituir processos manuais feitos em papel por uma plataforma digital mais organizada, prática e profissional.

A proposta é atender tanto a área administrativa quanto a área do aluno, oferecendo diferentes níveis de acesso e funcionalidades específicas para cada perfil.

## Funcionalidades principais

### Área administrativa
- Login de administrador
- Cadastro, edição e exclusão de alunos
- Upload de foto do aluno
- Cadastro de avisos
- Cadastro de treinos
- Cadastro de campeonatos
- Cadastro de links úteis
- Lançamento de presença
- Histórico de presença
- Relatório de frequência
- Dashboard com visão geral do projeto

### Área do aluno
- Login individual
- Visualização de avisos
- Visualização de treinos
- Visualização de campeonatos
- Visualização de links úteis
- Perfil com foto e dados cadastrais
- Consulta da própria frequência

## Tecnologias utilizadas

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Chart.js
